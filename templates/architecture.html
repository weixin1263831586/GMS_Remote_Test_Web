🔌 USB/IP 远程设备桥接

Windows (usbipd) → SSH → Test Host (vhci-hcd) → Android 设备

📋 USB/IP Protocol 工作原理

🖥️ Windows 端 usbipd-win 将 USB 设备共享到网络 (Port 3240)
🐧 Ubuntu 端 vhci-hcd 驱动创建虚拟 USB 设备，通过网络接收 USB 数据包
✨ 优势 远程设备访问 • 突破 USB 距离限制 • 多主机共享设备
🎯 应用 自动化测试 • 设备远程调试 • 分布式测试环境
